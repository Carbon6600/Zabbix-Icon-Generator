## ✨ Можливості / Features

### 🎯 Основні / Core
- **Автоматичне видалення фону** - BodyPix модель для точного виділення об'єктів
- **5 стандартних розмірів** - 24x24, 48x48, 64x64, 96x96, 128x128 (готові для Zabbix)
- **Розумне вписування** - 3 режими (contain, cover, fill) з налаштуванням відступів

### 🎨 Редагування / Editing
- Регулювання яскравості та контрасту
- Автоматичне покращення зображення
- Вибір кольору фону або прозорість

### 📝 Іменування / Naming
- Власна назва файлів
- Гнучкі роздільники (_ , - , пробіл)
- Автоматичне додавання розміру

### 📦 Вивантаження / Export
- Попередній перегляд всіх розмірів
- ZIP архів з усіма іконками
- PNG формат з прозорістю
